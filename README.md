# TeamCalen

![logo of TeamCalen](./teamcalen.svg)

## 项目简介

TeamCalen 是针对个人与小型团队而设计的日程管理解决方案，本项目是前端部分，采用 React 实现。

项目主要针对以下用户痛点进行设计：

### 个人

- 无法记录做过的事及要做的事；
- 纸质记录日程不易携带与保存；
- 日程表不能随时随地同步；
- 现有产品太复杂；
- 写了日程忘了做；
- UI 不够美观。

### 小型团队

- 日程表不能共享、协作修改；
- 不能随时修改日程表的查看权限；
- 日程涉及的文件分享困难；
- 不清楚项目的进度；
- 不清楚团队成员个人的进度；
- 日程表不能做权限分配。

- [x] 帐号的注册、登录与找回密码;

## 维护方式

项目采用双分支维护。

- 主分支版本应始终保持稳定可运行状态，且功能完整；
- 开发分支合并已稳定的功能。

新功能开发与 BUG 修复应当在新分支上进行，且每次 PR 的改动尽可能的小，以提高最终代码质量。
